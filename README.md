# 📖 Qari Flutter

**Qari** — мобильное приложение для изучения, чтения и заучивания Священного Корана.  
Приложение помогает пользователю ежедневно заниматься, отслеживать прогресс, слушать аудио, учить аяты по плану и развивать дисциплину.

---

# 🚀 Возможности приложения

## 👤 Регистрация пользователя
При первом запуске пользователь заполняет данные:

- Имя
- Фамилия
- Возраст
- Пол
- Уровень знаний
- Знание таджвида
- Уже выученные суры

---

## 🔐 Вход через Google

Поддерживается авторизация через Google аккаунт:

- Быстрый вход
- Автозаполнение имени
- Возможность облачного сохранения прогресса

---

## 📚 Режимы обучения

Пользователь выбирает один из режимов:

### 📘 Жаттау
Режим заучивания Корана.

Функции:

- Пошаговое изучение аятов
- Повторение
- План обучения
- Отметка "Жаттадым"

### 📖 Оқу
Режим чтения Корана.

Функции:

- Чтение аятов
- Прослушивание аудио
- Перевод
- Транскрипция

---
